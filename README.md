## Overview

Cyna is a full-stack e-commerce platform designed to allow customers to browse, purchase, and manage subscriptions to Cyna's cloud security services. The platform consists of:

- A **mobile-first web application** built with Vue 3 + Ionic
- An **Android application** powered by Capacitor, mirroring the mobile web experience
- A **REST API backend** built with Express, handling authentication, payments, and business logic
- A **PostgreSQL database** for persistent storage
- **Stripe** integration for secure payments and automatic subscription renewals
- An **admin back-office** for managing products, users, and orders

## Getting Started

### Prerequisites

- **Node.js** (v18 or higher recommended)
- **npm**
- A running **PostgreSQL** instance
- A **Stripe** account (with a secret key and a publishable key)
- A **ngrok** account and auth token

### Environment Variables

Copy `.env.example` to `.env` and fill in each value:

```bash
cp .env.example .env
```

| Variable | Description |
|---|---|
| `DATABASE_URL` | Connection string to your PostgreSQL database |
| `JWT_SECRET` | Secret key used to sign and verify JWT tokens |
| `STRIPE_SECRET_KEY` | Stripe secret key (server-side) |
| `VITE_STRIPE_PUBLIC_KEY` | Stripe publishable key (client-side) |
| `VITE_SERVER_URL` | Public URL of the backend, generated by ngrok |
| `NGROK_AUTH_TOKEN` | ngrok authentication token |

> **Note:** `VITE_SERVER_URL` should point to your ngrok tunnel URL. Start ngrok first (`npm run ngrok`), then copy the generated URL into your `.env` before starting the frontend.

### Running the App

Install dependencies:

```bash
npm install
```

Start the backend (with hot reload):

```bash
npm run server:dev
```

Start the frontend (Ionic dev server):

```bash
npm run serve
```

Start ngrok tunnel (in a separate terminal):

```bash
npm run ngrok
```

## Running on Android

**Requirements:** [Android Studio](https://developer.android.com/studio) must be installed.

Run the following commands in order:

```bash
# 1. Add the Android platform (first time only)
npx cap add android

# 2. Build the web app
npm run build

# 3. Sync build output to the Android project
npx cap copy android

# 4. Open the project in Android Studio
npx cap open android
```

Then run or build the app directly from Android Studio.

## Features

- **Homepage** — landing page presenting Cyna's services
- **Product & Category pages** — browse security solutions by category
- **Authentication** — register, login, and password reset flows
- **Account management** — update personal information and preferences
- **Checkout** — streamlined purchase flow with Stripe-powered payments
- **Subscriptions** — automatic renewal managed via Stripe
- **Order & subscription history** — view past and active orders
- **Admin back-office** — manage products, users, categories, and orders
- **AI Chatbot** — integrated assistant to guide users
- **Static pages** — legal notices, terms of service, contact form

## Contributing

Before opening a pull request, make sure your code passes all three checks:

```bash
npm run format      # Format code with Prettier
npm run lint        # Lint with ESLint
npm run type-check  # Type-check with vue-tsc
```

PRs that do not pass all three will not be reviewed.
